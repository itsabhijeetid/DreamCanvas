## Usage

Rename the `yo.env` file to `.env`.

Generate an API KEY at [OpenAI](https://beta.openai.com/) and add it to the `.env` file.

Install the dependencies

```bash
npm i express openai dotenv
```

```bash
npm i -D nodemon
```

```bash
npm install
```

Run server

```bash
npm start
```

or (if doesnt work)

```bash
npm run dev
```


Visit `http://localhost:5000` in your browser.
